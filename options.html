🔐 WordPress Password Generator

Tạo mật khẩu và hash WordPress an toàn cho việc quản lý website

Độ dài mật khẩu từ 8-50 ký tự (mặc định: 12)
Bao gồm ký tự đặc biệt như !@#$%^&*() để tăng độ bảo mật
PHPass tương thích với WordPress cũ, Bcrypt cho WordPress mới

ℹ️ Thông tin về WordPress Password:

  • PHPass: Thuật toán mặc định của WordPress, tương thích với tất cả phiên bản
  • Bcrypt: Thuật toán mới hơn, bảo mật cao hơn, dùng cho WordPress 6.8+
  • Salt: Mỗi password có salt riêng để tăng bảo mật
  • Iterations: Áp dụng 256 lần hash để chống brute force
  • Format: Hash có format $P$B[salt][hash] cho PHPass

📖 Hướng Dẫn Sử Dụng

🚀 Cách sử dụng nhanh:

  • Chụp ảnh có mũi tên: Chọn text hoặc click vào element, sau đó nhấn Ctrl+Shift+S
  • Chụp ảnh thủ công: Nhấn Ctrl+Shift+S không có vùng chọn để mở công cụ vẽ
  • OCR (Ảnh sang text): Right click → Công cụ → Ảnh sang text
  • Dịch tự động: Right click → Công cụ → Dịch tự động
  • WordPress Password: Right click → Công cụ → WordPress Password
  • Context menu: Right click trên bất kỳ trang web nào để xem các tùy chọn

⌨️ Phím Tắt

Ctrl+Z Hoàn tác trong công cụ vẽ
Esc Đóng công cụ vẽ

💡 Mẹo & Thủ Thuật

📸 Chụp ảnh màn hình:

  • Chọn text hoặc click vào element trước khi chụp để có mũi tên tự động
  • Nếu không có vùng chọn, công cụ vẽ sẽ mở để bạn vẽ thủ công
  • Trong công cụ vẽ, bạn có thể vẽ nhiều hình trên cùng một ảnh
  • Ảnh sẽ tự động được copy vào clipboard sau khi hoàn tất
  • Sử dụng màu đỏ để highlight quan trọng, màu khác cho ghi chú

🔍 OCR (Ảnh sang text):

  • Cần cấu hình Google Gemini API key trong tab "OCR Settings"
  • Model Gemini 2.5 Flash Lite là mặc định, nhanh và miễn phí
  • Hỗ trợ nhiều ngôn ngữ bao gồm tiếng Việt
  • Kết quả có thể copy trực tiếp vào clipboard
  • Thêm nhiều API keys để tránh rate limit
  • 👁️ Mới: Có thể xem lại và copy API keys đã lưu

🌐 Dịch tự động:

  • Tính năng riêng biệt, độc lập với OCR
  • Tự động OCR + Dịch text sang ngôn ngữ đích
  • Hỗ trợ 5 ngôn ngữ: Việt, Anh, Trung, Nhật, Hàn
  • 4 chế độ hiển thị: Inline, Window, Sidebar, Notification
  • Sử dụng: Right click → Công cụ → Dịch tự động

🔐 WordPress Password:

  • Mật khẩu: Tự động tạo với độ dài tùy chỉnh (8-50 ký tự)
  • Hash: Tạo hash WordPress tương thích với database
  • Thuật toán: PHPass (WordPress cũ) hoặc Bcrypt (WordPress mới)
  • Bảo mật: Salt riêng, 256 iterations, ký tự đặc biệt
  • Sử dụng: Copy password để đăng nhập, copy hash để update database
